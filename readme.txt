== Description ==
CNZZ&51LA for WordPress是用cnzz或者51la来统计WordPress博客访问量的插件。
以前在wordpress中用cnzz或者51la的统计功能，需要修改主题，换主题的时候，又要考虑修改主题，很麻烦，
有了CNZZ&51LA for WordPress这个插件之后，在wordpress用cnzz或者51la的统计功能就很方便啦。
它支持包括google analytics，yahoo统计在内的任意网站统计工具。
需要帮助，请访问
http://blog.printf.com.cn/cnzz-and-51la-for-wordpress/

== Installation ==
第一步，在http://blog.printf.com.cn/cnzz-and-51la-for-wordpress/下载CNZZ&51LA for WordPress；
第二步，将cnzz-and-51la-for-wordpress.zip上传到Wordpress插件目录里，通常就是“wp-content/plugins/”，然后解压缩；当然，也可以先解压然后上传；
第三步，在wordpress后台的“插件”菜单中，激活CNZZ&51LA for WordPress；
第四步，在cnzz或者51la上注册账号，获取统计代码；
第五步，在wordpress后台，点击“设置”菜单中的CNZZ&51LA for WordPress，把你的统计代码粘贴进去。
